# Training Planner

**Автор:** Юдина Екатерина Евгеньевна

## Описание
GUI-приложение на Python (Tkinter) для планирования и учета тренировок. Позволяет добавлять записи, валидировать корректность ввода, фильтровать историю по дате и типу занятий, а также автоматически сохранять данные в формат JSON.

## Инструкция по использованию
1. Запустите скрипт: `python main.py`
2. Заполните поля формы (Дата, Тип, Длительность) и нажмите **"Добавить тренировку"**.
3. Для поиска нужных занятий введите параметры в панели **"Фильтрация"** и нажмите **"Применить фильтр"**.
